FONTLOG for the CHAOS16 font

This file provides detailed information on the CHAOS16 Font Software.
This information should be distributed along with the CHAOS16 font
and any derivative works.

Basic Font Information

CHAOS16 is 16x16 pixel font that support ENG and RUS languages.

To contribute to the project you can contact me: reznicenkoroma@gmail.com


ChangeLog

23 July 2025 (Reznichenko Roman) CHAOS16 version 1.0
- Font created. ENG support and basic special symbols.

25 July 2025 (Reznichenko Roman) CHAOS16 version 2.0
- RUS support. Also added radioactive symbol (U+2622) for fun.
